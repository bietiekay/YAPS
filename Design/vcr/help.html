About YAPS

It's a HTTP Server that can be triggered to join a multicast group and forward the traffic from that multicast group directly to the client that asked for it using TCP.

The point in writing this software was to use the vast number of multicast streams in my network even in the wireless lan. And after I accomplished this goal there was the urge to extend YAPS beyond this functionality and make it a proper multicast vcr.

Visit my website: http://www.technology-ninja.com to get more information and updates on YAPS.

Skins

The layout is based on Dirk Jesse's amazing CSS-Framework YAML (Yet Another Multicolumn Layout © 2005-2006 by Dirk Jesse highresolution.info).
Creating your own YAPS-skin is a fairly simple thing. Just create a new subfolder in the skins directory of your YAPS-installation. The folder has to contain a "default.css" stylesheet and another subfolder named "images". As you can imagine, this folder should contain all images you want to use in your stylesheet ;)
As a first step, you may want to start with simply copying the default skin, rename it and start with modifying this stylesheet. Later release will include a complete documentation of all used classes and ID's. To make your start as easy as possible the default_new-skin contains all CSS files of the original YAML framework.
